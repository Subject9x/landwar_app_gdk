Unit Builder System

The following is for designing your own custom units in LANDWAR. You will assign stats to fill in the Unit Info card, then determine the total points cost for the unit. TAGS are optional upgrades that you can apply after the unit's base stats are set up. We strongly recommend just getting the official LANDWAR app, it has a Unit editor that will do all the cost calculations for you along with TAG choices and their costs.

Cost = (Size Value / 3)

  • Size influences several other core stats, such as Movement, and Armor.
  • Additionally, Size plays a key role in several TAG costs as well.
  • Smaller size units pay less for movement but more for armor.

Cost = ((Size Value / 1.5) * Evade Value) + (Move Value / 2).

  • Evade points are generated per-turn when a Unit moves 25% or more of their total Move Value.
  • Evade points are spent in the Combat Phse to boost DEF dice rolls, by increasing the roll result.

Cost = (DMG-Range * 4)

  • Amount of DMG a Unit will deal when attacking at range.
  • Primary attack type of the game, most units will have a ranged attack of some type.

Cost = (DMG-Melee / 2) + (Move Value / 4).

  • Amount of DMG a Unit will deal when attacking in Melee Range(0-1").

Cost = (Size Value + Move Value) / 2.

  • Movement value is the total inches a unit can move per-turn.
  • Movement also determines a Unit's Initiaitve Order, when during the Move Phase that they will activate.

Cost part 1 = (33% Armor Value ) * 2

Cost part 2 = (Armor Value - 33%) * 4

Sub total = (part 1 + part 2) - (Size / 2).

  • Armor is the primary health pool of every unit in the game.
  • Units suffer additional stress when Armor is below 50% of its starting value.
  • Larger size units pay less for armor.
  • Cost is weighted, the cost can be 0 but not negative!

Cost = (Move / 2) + (Range - 16) + (DMG-Ranged / 2).

  • Effective Range is in inches.
  • Effective Range is the range where targets will not gain Long Range defensive bonus.
  • LANDWAR does not have finite range limit, merely the 'best range' a unit attacks at.
  • Cost is weighted to 16" being the default attack range, the cost can be 0 but not negative!
Now take all the costs and total them up, this gives you the Base Cost of a unit in LANDWAR.
