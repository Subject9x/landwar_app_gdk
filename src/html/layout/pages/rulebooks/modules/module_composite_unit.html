LANDWAR

Module

Composite Unit

Composite Units allow players to pack multiple Unit Info Cards into a single model on the table. This module is best used to represent massive units with multiple sub-systems such as battleships, massive mecha, or AI-drive landships. To bind unit cards together, use the following contstruction rules, tag modifications, and gameplay changes. Like all modules, all players must agree to use this module before beginning play.

Begin by building each component as a standard unit.

  • 1 unit stat line is the 'base hull'.
  • The Size stats of other sub-components must add up to the Size of the base hull.
  • all components may have their own speed value, entire unit may make Standard moves using the highest Speed value.
  • Evasion cannot be used at all, this unit is massive!

  • Line of Sight
    • When targeting this unit, draw LoS to the model as normal, Attacker may then choose any unit card to resolve attacks against.
    • Making attacks with the Composite Unit, all components have the same LoS from the model to various targets and may resolve attacks separately.
  • Damage
    • Each component can be destroyed as normal, however entire Unit is not destroyed until all components have their armor reduced to 0.
    • Whenever a component is destroyed, Unit's Move becomes the remaining highest Move Value. If Unit is reduced to 0 Move, it is immobilized.
  • Stress
    • Components track their own stress as normal.
    • Whenever a component panicks, each other component suffers +1 Stress.
    • If 75% or more components panic, the entire unit panicks.
  • Objective Scoring
    • Only take the highest Armor value when determining Objective Secured.
