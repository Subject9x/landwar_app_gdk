LANDWAR

Module

Composite Unit

Limited-Use Items rules allow players to simulate special one-off abilities or weapons that a Unit might possess. These could be weapons with limited ammo, or single-shot abilities. LANDWAR mechanically isn't fully written to handle this kind of gameplay.

Each 'item' or 'weapon' will be its own Unit entry stat line.

  • Tags may be applied but must follow their normal requirements for mounting.
  • Each item/weapon must have a Size of 1 or higher.
  • Item info cards must then be declared applied to specific Units.
  • Total Size of all Items cannot exceed the Unit's Size value.
  • Each item info card represents a single 'use' of the item, attach multiple copies to a Unit to provide multiple uses, paying the costs accodingly.
  • The following tags cannot be placed on an Limited-Use Item:
    Cargo Courage-I Courage-II Crew-I Crew-II Fearless
    Fly Hero Mobile HQ Pack/Deploy Rank-Green Rank-Veteran
    Rank-Elite Stall Speed Transport Weak Rear Armor

  • Items with tags that effect the Initiative Phase must be activated and spent before the Initiative Roll.
  • Tag Interactions
    • Heavy Armor: can be used to reduce damage, but if damage in the Resolution Phase would exceed Unit's Armor, Unit is destroyed.
    • Self-Healing: Max Armor regained is limited to the item's Size value.
