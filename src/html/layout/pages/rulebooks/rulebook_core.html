Initiative Phase

Players determine who will activate their first units during the following phases.

  1. Each side rolls 1D6 and compare the results.
    • Each player rolls their own Initiative regardless of any alliances
    • Any non-destroyed Unit may use TAGS applicable to this phase.
  2. Side with higher total is the Offense.
  3. Side with lower total is the Defense.
  4. IF the totals are tied, Defense player from last turn becomes Offense Player.
  5. Offense Player wins activation ties during the Movement Phase and moves those units after the Defense Player.
  6. Resolve any Initiative effects or events now.
  7. Panicked - when Unit starts phase with this token:
      8. remove 1 Stress Point.

Movement Phase

Players take turns moving Units during this phase.

Units resolve their movement in the following sequence.

  1. All Units move in order of their Ini Value lowest to highest.
  2. Ini Value is a Unit's current MV stat after applying any modifiers.
  3. Defense must move Units first when the Ini Values are tied.
  4. When moving a Unit, the controlling Player can select one of the following options:
    • Declare Stationary.
      • Unit may Rotate to a new Facing.
      • Unit gains +1 ATK for this Combat Phase.
      • Unit cannot use its Evade bonus for this Combat Phase.
      • Unit suffers -1 DEF on this Combat Phase.
    • Make Normal move.
      • Unit may move in any direction up to to their current Move value.
      • Unit cannot move through an enemy Unit.
      • Unit may change their Facing at the end of the move.
      • Unit may use its Evade value IF it moves more than 25% of its total Move value.
    • Use TAG rules, if Players are using TAG special rules for units, and a Unit has a TAG that allows specific movement, it can be used here.
  5. Complete selected Movement and check for the following:
    • Terrain.
    • High Speed.
    • Danger Close.
    • Overwatch.

Unless otherwise specified, Units move 1/2 their Move value when:

  • Moving through intervening terrain.
  • Starting move inside terrain.

Whenever a Unit completes a Normal move where the total distance moved was greater than 11", the Unit has completed a High Speed Move.

Units during their this Combat Phase are given the following effects.

  • ATK - 2. (Minimum of 1).
  • re-roll 2 DEFdice.
  • When a Unit starts their Move 1" or less from at least 1 Enemy Unit.
    • Unit may only move half their current Move value.
    • One enemy unit may make a free Overwatch attack against the Unit but can only be a Melee Attack.

Enemy Unit can make an immediate Ranged Attack at -2 ATK with the following conditions:

  • Panicked Units cannot make Overwatch attacks.
  • When a Unit moves from the Forward arc to the Rear arc of an Enemy Unit.
  • IF the Unit was in Target's Line of Sight at the start and end of the move.
  • IF the Unit ends their Move 6" or less from the Target Unit.

Combat Phase

Offense Players resolve all their Attacks first, then Defense players.

Attacks are split into 2 categories - Range and Melee.

Damage from all Attacks are not applied until the Resolution Phase!

Ranged Attacks

Requirements:

  • Attacking Unit is not currently in Melee Range(1") of any Enemy Unit.
  • Target must be the closest Unit to the Attacker in Line of Sight.
    • See Focus Fire below this section.
  • Attacking Unit must have Line of Sight to the Target Unit.
  • TAG rules may override, add, or change these qualifications!

Melee Attacks

Requirements:

  • Unit must have a Melee Damage stat greater than 0.
  • Target must be in Melee Range of Unit. - default 0-1".
  • Target must be in the Attacker's Forward Arc.

There are only 3 main Range Brackets in the game.

  • Melee - 0" to 1".
  • Effective Range, default is 16".
    • Units can have their own Effective Range using the Range stat on their Info Card.
  • Long Range, unlimited distance.
    • Target gains +1 DEF dice for this defense roll.

When a Unit makes a Ranged Attack they can select the non-closest valid Target under the following limits:

  • Unit cannot be Panicked.
  • Unit Stress level must be below the Unit's max. - default is 6
  • IF closest enemy Unit is in Effective Range, and desired Target is at Long Range:
    • Unit suffers +2 Stress for each attack that misses.
  • IF closest enemy Unit is under 6" and desired Target is > 6" range:
    • Unit suffers +1 Stress for each attack that misses.
  • IF added Stress would trigger a Stress Check roll for one before Attacking.
    • If the Unit fails the Stress check they may not attack this Phase.
    • Unit is not Panicked however, Player must check again in Resolution Phase.

RANGED Attacker Dice Pool

  • Starting Amount: 3D6.
  • If Unit has Panic Token:
    • total Dice is halved and minimum of 1.
  • Stationary: Attacker gains +1 ATK dice.
  • Check Short Range.
  • Apply any TAG rules to the ATK amount.

When the Target is 1" to 4" away.

  • Attacker gains +1 ATK.

RANGED Defender Dice Pool

  • Starting Amount: 3D6
  • If Unit has Panic Token:
    • total Dice is halved, minimum of 1.
  • Check Cover.
  • Check Evasion.
  • Check High Speed Move.
  • Apply any TAG rules to the DEF amount.

When the Target is at Long Range

  • Target may +1 DEF dice to this roll.

If Defender is behind or in a board object agreed as 'Cover':

  • Defender may re-roll half their total dice, but object is removed from board

If Target was not Stationary during this Turn's Movement Phase

If Target moved 1/4(25%) or more of their current MOVE.

  • Target may add up to unit's EVADE value to their DEF roll results.
  • Evade points are spent per Turn, once they're all spent, they only recharge if Unit moves next Turn!

If Attacker moved 12" or more during this Turn's Movement Phase:

  • Defender may re-roll up to 2 DEF dice.

MELEE Attacker Dice Pool

  • Starting Amount: 4D6
  • If Unit has Panic Token:
    • total Dice is halved and minimum of 1.
  • Stationary: token gives Attacker +2 ATK.
  • Apply any TAG rules to the ATK amount.

MELEE Defender Dice Pool

  • Starting Amount: 3D6
  • If Unit has Panic Token:
    • total Dice is halved, minimum of 1.
  • Check Evasion.
    • Stationary: Defender cannot use Evasion stat.
  • Check High Speed Move.
  • Apply any TAG rules to the DEF amount.

If Target was not Stationary during this Turn's Movement Phase:

  • Target may add up to unit's EVADE value to their DEF roll results.

Resolve Attacks

Now Players assemble the dice they will need to roll to see if the Attack is successful or not.

Damage Phase

Now both Attacker and Defender roll their pools of D6s.

Attacker: each 4+ is 1 'success'.

Defender: each 5+ is 1 'success'.

Compare the two rolls:

  • For Each Defender Success, remove 1 Attacker Success.
  • If Attacker has 1 or more Success remaining.
    • Defender suffers an amount of DMG equal to the Attacker's DMG value for that Range.
  • Check Target Destroyed.
  • Check Heavy Fire.
  • Check Crew Shaken
  • Check Focused Fire.

When the Defenders Armor stat is reduced to 0:

  • The unit is now considered Destroyed.
  • Do Not remove the unit until the Resolution Phase of this Turn.
  • The Destroyed unit may still make their final attacks during this Combat Phase.
  • Each remaining friendly Unit in LoS and 6" suffers +1 Stress point.

For 1 uncancelled ATK hit of 6:

  • Defender suffers +1 Stress Point.

If a Unit ends the Damage Phase with any Armor damage at or below 50%:

  • Unit suffers +1 Stress Points.

If a Unit ends the Damage Phase with any Armor damage at or below 75%:

  • Unit suffers +1 Stress Points.

If Defender was attacked (regardless of hits) by 3 or more enemy Units this turn:

  • Defender suffers +1 Stress Point.

Additional Attacks

Players continue to make and resolve attacks as needed or declared until any-or-all Units that can attack, have done so.

Resolution Phase

All damage from the Combat Phase is applied now.

Units that have been destroyed are removed at this time.

  • Check Panic and Stress.
  • Check for Victory Conditions.

Any non-Destroyed Unit with 1 or more Stress Tokens must make a Stress Check.

  • note: TAGS may be applied for Stress Checks and tokens at this time.
  • Controlling Player rolls 1D6 for each of their Units with Stress Tokens.
  • If the roll is equal or higher than the current Stress Tokens for that Unit:
    • Player removes all Stress Tokens from the Unit.
  • If the roll is less than the number of Stress Tokens for the Unit:
    • Unit has become Panicked.
    • Place a Panicked token next to the Unit or on its data card.

At the end of the entire turn, players will check if the Victory Conditions for the match have been satisfied.

  • If conditions are met, the game is over!
  • If conditions are not met, proceed back to the Initiative Phase to begin a new Turn.

REFERENCE

Finally, at the end for reference are some important concepts that are used during gameplay.

TAGS are an optional set of special rules for enhancing Units in your army.

  • A TAG will declare when it can be used or applied.
  • Each Unit can only use each of its TAGS once per Turn.
  • Adding and Subtracting dice to a roll must happen before any rolls are made.
  • Each dice in a situation can only be re-rolled ONCE.
  • Keyword: must re-roll; Player must re-roll all dice up to the number required.
  • Keyword: may Player can choose a number of dice to re-roll up to the set total.
  • Dice Modifiers are applied AFTER re-rolls.

Primary method to determine if a Unit can see other Units, usually for Attacks.

Basic Method

  • Draw a line from the center of the model of the Unit.
  • to the center of the model of any target Units.
  • If the line is not obstructed by enemy models or Terrain Objects, then Line of Sight is achieved.

Advanced Methods(OPTIONAL)

  • Compare the [Size] values of all models that are between the Unit and a target.
  • If the Unit's [Size] value is greater than the blocking model, then Line of Sight is achieved.
  • If the Target's [Size] value is greater than the blocking model, then Line of Sight is achieved.

Terrain for LANDWAR can be varied and complex, its completely up to the players and materials they have available.

Basic Rules

  • For each Terrain Object Players must agree on its general properties:
    • Does the object block Line of Sight?
    • Can Units move into and out of the object?

Advanced Rules(OPTIONAL)

  • For each Terrain Object Players must assign properties:
    • [Size] value, used with Line of Sight advanced method.
    • [Armor] value (OPTIONAL), allows Terrain Object to be attacked like a normal Unit and destroyed.
