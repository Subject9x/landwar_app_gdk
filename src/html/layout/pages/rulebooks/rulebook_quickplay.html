LANDWAR Quick Play

Core Rules

Turn Order
  1. Initiative
  2. Movement
  3. Combat
  4. Damage
  5. Resolution
  • Determine order for turn.
  • alternate moving 1/2 of eligible units.
  • alternate making attacks.
  • apply any damage from successful attacks.
  • remove destroyed units, check end game.
note: where applicable, TAG rules override the core rules of the game.
1. Initiative
  • Players roll 1D6 and add any INI TAG mods and compare results.
  • Highest value Player becomes ATTACKER for turn.
  • Lowest value Player becomes DEFENDER for turn.
2. Movement
  • Players take turns moving 1/2 of their force at a time.
  • DEFENDER makes the first move.
  • Stationary: place token, Unit can rotate in place.
  • Normal: Unit may move up to its current total MV value.
  • TAG: make move as described in TAG rule.
  • IF Panicked remove 1 Stress point.
3. Combat
  • Any unit with Melee or Ranged values > 0 may make 1 attack.
  • Check Line of Sight: Enemy units block LoS, terrain may block Line of Sight.
  • Ranged Attack: check for Effective Range vs Long Range
  • Melee Attack: 0-1" range and facing enemy.
  • Default Attack(ATK) dice pool is 4.
  • Default Defense(DEF) dice pool is 3.
Combat Modifiers
  • Attack: Stationary +1 ATK
  • Attack: Panicked halve base ATK
  • Attack: High Speed -2 ATK and re-roll 2 DEF.
  • Attack: Overwatch -2 ATK
  • Defense: Long Range +1 DEF
  • Defense: Moved more than 1/4 MV - may spend Evade points on DEF rolls.
  • Defense: Cover - may re-reroll half base DEF dice.
4. Damage
  • Each attack the Attacker and Defender roll their D6 pools.
  • Attacker counts each roll of 4+
  • Defender counts each roll of 5+
  • When Attacker success total is greater than Defender total, Target Unit takes all the Damage from the attack.
  • subtract Damage from Target's Armor stat first, then when Armor is 0, subtract from Structure.
  • Ranged Attack: missed shots at non-closest Target +1 stress at Effective Range OR +2 at Long Range.
5. Resolution
  • Any Units reduced to 0 Structure in the Damage phase are now marked as Destroyed.
  • Anytime a Player loses 1 or more Units in a Turn, all of their remaining Units suffer +1 Stress.
  • Any Units with more than 0 Stress tokens must make a Stress Check.
  • Any Units with a Panicked token must make a Stress Check.
Stress
  • +1 Stress for each attack that hits Armor when Armor is below 1/2 total.
  • +2 Stress if Unit took any Structure damage this Turn.
  • +1 Stress if Unit was target of 3 or more attacks this Turn.
  • Any TAG rules that apply as well.
Stress Check
  • Each Unit with 1 or more Stress Tokens rolls 1D6:
  • If result greater than total Stress Tokens, remove all tokens.
  • If result equal or less than total Stress Tokens, give unit 1 Panicked token.
